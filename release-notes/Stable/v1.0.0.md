# Release Notes for v1.0.0

Channel: Stable
Status: Released
Date: 2026-07-09
Package: Arbor

## Summary

Arbor `v1.0.0` is the first stable public package release.

This release publishes Arbor as a standalone Luau package for compile-time
typing over analyzer-visible Roblox data model trees. It focuses on typed
child-name surfaces, class-filtered child helpers, and a small runtime loader
for validated direct-child ModuleScripts.

## Improvements

| Note | Status |
| --- | --- |
| Publishes Arbor as a standalone Luau package. | Released |
| Adds analyzer-facing helpers for direct child names, child records, child lookup, and class-filtered children. | Released |
| Adds `LoadModuleMap.From(root, validate)` for validated runtime loading of direct ModuleScript children. | Released |
| Exports package assets as `Arbor.rbxm`, `src.zip`, and `src.tar.gz`. | Released |

## Notes

- Arbor has no required peer package dependencies.
- Examples remain repository-only and are not included in exported package assets.
- Runtime discovery and compile-time type visibility are separate concerns.
- Rojo may provide analyzer-visible source-map data, but Arbor does not require Rojo specifically.
