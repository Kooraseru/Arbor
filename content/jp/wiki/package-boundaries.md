# パッケージ境界

Arbor が答えるのは：

```txt
Luau アナライザーは、所有されたインスタンスサーフェスについてどんな静的構造を知っているか？
```

Arbor が答えないのは：

```txt
どんなランタイムモジュールが存在するか
いつ起動が起きるか
どの順番で挙動が実行されるか
値がネットワークをどう渡るか
権限つきアクションがどう公開されるか
どんな意味上の概念が存在するか
```

## 許可する責務

```txt
型関数
型付き lookup の慣習
必要な場合の生成済み型サーフェス
アナライザー向け ID
小さなランタイムローダーヘルパー
```

## 許可しない責務

```txt
ランタイムレジストリ
起動順序制御
パイプライン実行
シリアライズタグ
ディスパッチ解析
ディスクリプター identity
外部パッケージ lookup
```

`RuntimeLoaders` が許可されているのは、動的ロードと検証を組み合わせる小さなヘルパーだからです。レジストリやライフサイクルパッケージに育てないでください。
