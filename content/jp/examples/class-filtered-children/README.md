# クラスで絞り込む子の例

この例は、Arbor が直接の子をクラスで絞り込めることを確認します：

- `ChildrenOfClass<typeof(script), ModuleScript>` は、条件に合う直接の子インスタンスのテーブル形状を返します。

スクリーンショットの対象：

- Explorer ツリー。ワークスペース内に `ModuleScript` の子と、少なくとも 1 つの `ModuleScript` ではない直接の子がある状態。
- `ModuleChildren` が書かれている `init.luau`。
- アナライザーが `Kick` と `Ban` を、クラスで絞り込まれた子インスタンスのキーとして受け入れること。
- ワークスペースで一時的に試したとき、`ModuleScript` ではない子のキーをアナライザーが拒否すること。

チェックイン済みのファイルシステム版には、`ModuleScript` の子が含まれています。拒否されるケースのスクリーンショットを撮るときは、Studio かワークスペースビューで一時的に `ModuleScript` ではない直接の子を追加し、アナライザー結果を撮ってから、そのローカルのテストオブジェクトを捨ててください。
