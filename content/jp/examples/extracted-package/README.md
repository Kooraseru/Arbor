# 抽出済みパッケージの例

この例は、Arbor が抽出済みパッケージとして使われても同じように動くことを確認するためのチェックリストです。

期待するワークスペース形状：

```txt
WorkspaceOrPackageRoot
  Packages
    Arbor
      src
  Examples
    ExtractedConsumer
      init.luau
      Kick.luau
      Ban.luau
```

チェック項目：

- 利用側が、抽出済みパッケージの場所から Arbor を `require` している。
- ルートファサードの使い方が動く。
- 用途別モジュールの使い方が動く。
- アナライザー結果がリポジトリ内の例と一致する。
- ランタイムローダーの検証が、引き続き `ModuleScript` の戻り値の型を受け持っている。

このフォルダーはいまのところドキュメント用だけです。抽出済みパッケージのパスは、使う側のワークスペースやパッケージマネージャーに依存するためです。エクスポート時には、公開配布向けに選んだ具体的なワークスペース構成を追加してください。
