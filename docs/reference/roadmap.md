# Roadmap

Arbor grows only when analyzer behavior can support the API.

## Near-Term

- keep examples checked against the current Luau analyzer
- refine RBXM export and release artifacts
- document exact analyzer expectations
- add screenshots only after the docs prose is stable

## Possible Later Work

- more package extraction checks
- more complete analyzer fixture coverage
- additional helper types if probes prove they are stable

## Not Planned

- runtime lifecycle systems
- registries
- recursive dynamic package discovery
- broad framework behavior
